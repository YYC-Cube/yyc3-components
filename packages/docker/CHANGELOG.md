# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-26

### Added

- 🎉 Initial release of @yyc3/docker
- ✨ Complete container management (create, start, stop, restart, remove)
- ✨ Image management (pull, remove, build)
- ✨ Network management (create, remove, connect)
- ✨ Volume management (create, remove, mount)
- ✨ Container logs viewing (real-time and historical)
- ✨ Docker metrics (CPU, memory, network, I/O)
- 🎨 ContainerManager component
