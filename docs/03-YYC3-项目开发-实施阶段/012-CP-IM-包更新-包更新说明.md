# 包更新说明

## 新增包

1. **@yyc3/websocket** (v1.0.0) - WebSocket 连接管理
   - useWebSocket Hook
   - 自动重连和心跳
   - 消息订阅系统
   - 连接统计监控

2. **@yyc3/ai** (v1.0.0) - AI 对话功能
   - useAI Hook
   - 流式响应处理
   - 多提供商支持
   - 配置持久化

3. **@yyc3/terminal** (v1.0.0) - 终端会话管理
   - useTerminal Hook
   - 会话管理
   - 命令执行
   - 历史记录

4. **@yyc3/diagnostics** (v1.0.0) - 系统诊断工具
   - 系统健康检查
   - 数据验证和修复
   - 格式化报告

5. **@yyc3/chat** (v1.0.0) - 聊天组件和 Hooks
   - useChatPersistence Hook
   - useChannelManager Hook
   - TypingIndicator 组件
   - 数据导入导出

## 更新的包

**@yyc3/utils** (v1.0.0 → v1.1.0)
- 新增验证工具函数
- isString, isNumber, isBoolean 等
- isEmail, isURL 格式验证
- deepMerge, cleanObject 工具

## 总计

- **总包数**: 12个（原8个 + 新增4个）
- **总 Hooks**: 15个（原7个 + 新增8个）
- **总组件**: 60个（原52个 + 新增8个）
- **总工具函数**: 38个（原28个 + 新增10个）
