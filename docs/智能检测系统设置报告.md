## 🚀 自动化流程

### Pre-Push Hook 流程

```
git push origin main
    ↓
Pre-Push Hook 触发
    ↓
运行智能检测 (smart-check.sh)
    ↓
┌─────────────┐
│ 检测通过?   │
└──────┬──────┘
       ├─ 是 → ✅ 允许推送
       └─ 否 → ❌ 阻止推送
                 ↓
            显示错误详情
                 ↓
            提供修复建议
```

### 智能提交流程

```
smart-commit.sh
    ↓
智能检测
    ↓
┌─────────────┐
│ 检测通过?   │
└──────┬──────┘
       ├─ 否 → 智能修复(最多3次)
       │        ↓
       │    重新检测
       │        ↓
       │   ┌─────┴─────┐
       └─ 是         是
         ↓           ↓
    CI/CD 验证   创建提交
         ↓           ↓
    ┌─────┴─────┐   ↓
   通过?       是  推送
    ↓           ↓
   否 └→ 重新修复 → 完成
```
